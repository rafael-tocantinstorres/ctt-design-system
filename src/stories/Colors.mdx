    ## Usage Guidelines

    <div className="usage-note">
      <h4>Color Token Structure</h4>
      <p>Our color system follows a three-tier structure:</p>
      <ul>
        <li><strong>Brand Colors:</strong> Raw color values that define the brand palette</li>
        <li><strong>Core Colors:</strong> Semantic assignments that give meaning to colors</li>
        <li><strong>Base Colors:</strong> Component-level colors for specific UI elements</li>
      </ul>
      
      <h4>Implementation</h4>
      <p>Always use the CSS custom properties (tokens) rather than hardcoded hex values. This ensures consistency and makes it easy to maintain and update the design system.</p>
      
      <h4>Accessibility</h4>
      <p>All color combinations in this system maintain proper contrast ratios for accessibility compliance. When creating custom combinations, always verify contrast ratios meet WCAG guidelines.</p>
    </div>
